📚 Algorithm Explanation

Quick Sort

Quick Sort is a Divide and Conquer algorithm that picks a pivot element and partitions the array around it. Elements smaller than pivot go to left, larger to right, then recursively sorts the subarrays.

Steps:
1. Choose a pivot (last element)
2. Partition the array
3. Recursively sort left and right halves

KMP Algorithm

KMP (Knuth-Morris-Pratt) is an efficient string matching algorithm that uses a preprocessing LPS (Longest Prefix Suffix) array to avoid redundant comparisons.

Steps:
1. Build LPS array from pattern
2. Use LPS to skip characters while matching
3. Find all pattern occurrences in O(n+m) time
